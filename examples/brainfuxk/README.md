## Brainfuxk Interpreter

This example is the Brainfuxk Interpreter written by C language.

### Build
```
$ ../../711cc -o bfi.o bfi.c
$ gcc -o bfi bfi.o
```

### Run brainfuxk
Run HelloWorld:
```
root@a9a608a739a4:/home/examples/brainfuxk# ./bfi hello.bf
Hello World!
```
